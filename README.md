# Digi Cook server

## Utilisation
``` bash
# Installation des dependences npm.
npm install
# Créer les tables de la BDD.
npm run createTables
# Créer la base de données et ajouter les données de test.
npm run load

# Lancer le serveur en development.
npm run dev
# Build le server pour la production
npm run build
# Lancer le server en production
npm run start
```
